What does the app do?


  • Aims to help the elders and caregivers in controlling their health and performance.
  • Uses alarm as a feature for reminding the users when to take medications, sleep, eat and wake up.
  • Entertains users from depression using different entertaing acctivities
  • Solves many problems faced by elders and caregivers as most of them lack someone to look after.
  • Monitors almost all activities of users well.


    • Characterstics of the app


    • Cross platform: The application works multiple platforms, and this would increase the flexibility of the application. For example, it runs in androids and IOS platforms.
    • Interface: The application is user friendly in which the user should easily understand the application. It also include pictures and graphics. It inspires users to follow up the guides stated in the application in the most understandable way.
    • Ease of use: The application is so simple and avoid the complexity to use. The application also uses simple words to understand so that you can easily use the app to follow up the guides.
    • Usage offline: The application works offline so that users could not encounter any problems while their Wi-Fi is off or if they are in places without internet connection. There would be a requirement of the internet only if we want to update the application only.
    • Integration: The application also works with integration of other devices like the smart watch and others. This would help the users to track their progress.
    • Ease Data Entry: the application receives information from the users without pressuring them. The users should enter the information easily without making it complex in the form provided to them through the app.
    • Personalized information: the application is able to receive personal information of the users and can provide the services according to the information related to specific people.
    • Data confidentiality and security: The data provided is confidential and private. The record of data provided by the users is secured and not shared by the application as ads to other people.
    • Articles: this application posts some articles that users can visit and learn from it. The articles provide guidelines on how to use it and the success of the application usage by providing examples of other people using the app and successfully accomplishing their instructions.
